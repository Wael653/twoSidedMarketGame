{% blocktranslate %} Spielübersicht:
  • Das Spiel wird von zwei Spielern gespielt:
    • Spieler A betreibt Flohmarkt A
    • Spieler B betreibt Flohmarkt B
  • Es gibt insgesamt 100 Verkäufer und 1000 Käufer, die sich auf beide Märkte verteilen (nicht unbedingt vollständig verteilt).
{% endblocktranslate %} {% blocktranslate %} Ziel:
  • Beide Spieler versuchen, über die gesamte Spielzeit ihren Gesamtumsatz zu maximieren.
{% endblocktranslate %} {% blocktranslate %} Spielablauf:
  • Das Spiel läuft rundenbasiert ab.
  • Die Anzahl der Runden wird auf der Konfigurationsseite festgelegt.
  • Dort wählt man auch eine Kombination der Netzwerkeffekte aus.
{% endblocktranslate %} {% blocktranslate %} Wechselseitiger Effekt:
  • Ein zentraler Aspekt in einem zweiseitigen Markt und beschreibt, wie sich die beiden Marktseiten (Verkäufer und Käufer) gegenseitig beeinflussen:
    • Wenn mehr Verkäufer auf dem Markt sind, wird dieser für Käufer attraktiver (größere Auswahl).
    • Wenn mehr Käufer da sind, wird der Markt für Verkäufer attraktiver (mehr potenzielle Kunden und höhere Umsätze).
{% endblocktranslate %} {% blocktranslate %} Ablauf einer Runde:
  • Beide Spieler legen nacheinander folgende Preise fest, ohne die Preise des anderen Spielers zu kennen:
    • Standmiete für Verkäufer
    • Eintrittsgebühr für Käufer
  • Anschließend werden folgendes berechnet:
    • Anzahl der Verkäufer auf jedem Markt
    • Anzahl der Käufer auf jedem Markt
    • Umsätze der beiden Spieler in dieser Runde
  • Am Ende der Runde sehen beide Spieler:
    • ihre eigenen gesetzten Preise und Ergebnisse
    • die Preise und Ergebnisse des anderen Spielers
    • ein Balkendiagramm, das die Informationen der aktuellen Runde zusammen mit denen der vorherigen Runden anzeigt
  • Mit diesen Informationen setzen die Spieler in der nächsten Runde neue Preise fest.
{% endblocktranslate %} {% blocktranslate %} Spielende:
  • Nach der letzten Runde werden die Gesamtumsätze beider Spieler angezeigt.
  • Der Spieler mit dem höheren Gesamtumsatz wird als Gewinner gekürt.
{% endblocktranslate %}
